## Wallet

Represents a wallet.

| MIME type                                 |
|-------------------------------------------|
| application/vnd.iris.billing.wallet+json |

| Name               | Description                                  | Type                                    |
|--------------------|----------------------------------------------|-----------------------------------------|
| id                 | the wallet id                                | string                                  |
| balance            | the wallet balance                           | double                                  |
| lastChange         | the last modification date                   | DateTimeOffset                          |
