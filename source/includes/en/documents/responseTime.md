## ResponseTime

Represents a response time.

| MIME type                                 |
|-------------------------------------------|
| application/vnd.iris.response-time+json |

| Name                     | Description                                    | Type                         |
|--------------------------|------------------------------------------------|------------------------------|
| timeUnit                 | the time unit (Miliseconds, seconds, minutes)  | TimeUnit                     |
| value                    | the response time                              | long                         |
