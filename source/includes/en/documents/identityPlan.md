## IdentityPlan

Represents a plan for the identity.

| MIME type                                 |
|-------------------------------------------|
| application/vnd.iris.billing.identity-plan+json |

| Name                     | Description                                    | Type              |
|--------------------------|------------------------------------------------|-------------------|
| ownerIdentity            | the identity of the account                    | Identity          |
| planId                   | the id for a plan                              | string            |
