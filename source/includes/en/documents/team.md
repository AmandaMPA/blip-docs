## Team

Represents an agent attendant team.

| MIME type                                 |
|-------------------------------------------|
| application/vnd.iris.desk.team+json |

| Name                     | Description                                    | Type                         |
|--------------------------|------------------------------------------------|------------------------------|
| name                     | the name of the team                           | string                       |
| agentsOnline             | the number of online agents on the team        | int                          |
