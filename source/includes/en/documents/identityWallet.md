## IdentityWallet

Represents a wallet for the account.

| MIME type                                 |
|-------------------------------------------|
| application/vnd.iris.billing.identity-wallet+json |

| Name                     | Description                                    | Type              |
|--------------------------|------------------------------------------------|-------------------|
| ownerIdentity            | the identity of the account                    | Identity          |
| walletId                 | the id for the wallet                          | string            |
